### 로그인아이디의 변화가 없다면 중복체크 요청하지 않도록

**1\) checkLoginIdDup()**

```javascript
function checkLoginIdDup(el) {
		const form = $(el).closest('form').get(0);
		
		if (form.loginId.value.length == 0) {
			validLoginId = '';
			return;
		}
		
		if (validLoginId == form.loginId.value){
			return;
		}
		
		$('.loginId-msg').html('<div class="mt-2">확인중...</div>');		
```
